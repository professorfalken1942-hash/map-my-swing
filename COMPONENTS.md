# Golf Swing Analyzer – Components

## Core Components

### RecordButton

* Large primary CTA
* States: idle, recording, stop

### VideoPlayer

* Play / pause
* Scrubber (frame control)
* Slow motion toggle

### PoseOverlay

* Draw skeleton on body
* Highlight hips + shoulders
* Optional: fade non-critical joints

### MetricCard

* Title (e.g., "Hip Rotation")
* Value (angle or timing)
* Status (good / needs work)

### FeedbackCard

* Short sentence
* Optional icon (warning/check)

### SessionListItem

* Thumbnail
* Date
* Quick score (optional)
