# Map My Swing — Golf Swing Analyzer

A premium golf swing analysis platform. Upload videos, get frame-by-frame breakdown, tempo analysis, and actionable insights.

## Stack

- **Frontend:** Next.js 16, React 19, TypeScript
- **Styling:** Tailwind v4, custom CSS
- **Hosting:** Vercel
- **Design:** Gold/cream/dark theme with premium aesthetic

## Features (MVP)

- Video upload (drag & drop)
- Frame-by-frame playback controls
- Swing metrics display (tempo, club speed, plane, path)
- Insights panel with recommendations
- Mobile-responsive layout

## Future Features

- AI swing analysis (MediaPipe pose estimation)
- Swing comparison (side-by-side)
- Swing history and tracking
- Pro tips and coaching
- Share and social features

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deployment

```bash
vercel --prod --yes
```

## Design System

- **Colors:** Gold (#d4af37), Cream (#f8f6f3), Dark (#1a1a1a)
- **Spacing:** 8pt grid
- **Typography:** System fonts, 600 weight for headings
- **Corners:** Sharp (0px border-radius) for premium feel
