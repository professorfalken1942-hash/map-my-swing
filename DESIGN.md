# Golf Swing Analyzer – Design System

## Design Vision

A calm, focused experience that removes noise and helps golfers improve without thinking too hard.

## Tone

* Confident
* Supportive
* Simple

## Color System

* Background: dark or muted neutral
* Primary accent: single highlight color (used sparingly)
* Feedback:
  * Green = good
  * Yellow = needs work
  * Red = problem

## Typography

* Large, readable text
* Minimal font variations
* Clear hierarchy (Title → Metric → Feedback)

## Layout

* Top: video player
* Middle: overlay (skeleton)
* Bottom: metrics + feedback

## Spacing

* Generous padding
* Avoid dense layouts
* Each section clearly separated

## Motion

* Subtle transitions only
* Smooth scrubbing for video
* No distracting animations

## Iconography

* Simple, minimal icons
* Used only to support meaning (not decoration)

## UX Patterns

* One task per screen
* Clear primary CTA
* No hidden actions

## Anti-Patterns (Avoid)

* Data overload
* Complex charts
* Tiny text
* Multiple competing CTAs
